# 🎵 Soolking - African Jungle Music Card

A sleek, dark-themed music album UI component inspired by the discography of French artist Soolking. 

## ✨ Design & Layout Features
* **Immersive Dark Mode:** Built with a rich `#0B0B0C` deep charcoal background to mimic a real modern streaming app interface.
* **Responsive Media Transitions:** Uses fluid media queries (`@media (min-width: 768px)`) that cleanly transform a stacked mobile layout into an aligned desktop row element using Flexbox.
* **Digital Stadium Typography:** Features imported Google Fonts using `Anton` for impactful headings and `Syne Mono` for an authentic tech/music feel on interactive elements.

## 🛠️ Tech Stack
* **HTML5:** Structured semantic markup including linked custom browser icons (`shortcut icon`).
* **CSS3:** Logic-first custom layout configuration using `flexbox`, responsive viewport breakpoints, and custom box-shadow design.
